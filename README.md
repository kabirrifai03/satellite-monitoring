# Satellite Monitoring System

This project is a satellite monitoring system developed using React for the frontend and Flask for the backend.

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   cd backend
   pip install -r requirements.txt
   cd ../frontend
   npm install
3. Run the backend:
    cd backend
    python app.py
4. Run the frontend:
    cd frontend
    npm start


Dengan struktur dan kode di atas, Anda dapat memulai proyek Anda dengan baik. Pastikan untuk mengimplementasikan logika spesifik sesuai kebutuhan Anda, terutama pada bagian parsing Excel dan visualisasi D3.js. Selamat mengerjakan skripsi!
